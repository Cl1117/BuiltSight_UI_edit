How to run this website quickly. (Windows)

1. Open "BuiltSight_UI_edit.sln" in Visual Studio Community.
2. Open Terminal (Ctrl + `).
3. Type in command "cd BuiltSight_UI_edit\clientapp".
4. You will see "clientapp>".
5. Type in command "nmp start".
6. The website will show up in "http://localhost:3000/" automatically.
